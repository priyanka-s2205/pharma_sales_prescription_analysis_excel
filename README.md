Pharmaceutical Sales & Prescription Analysis (Excel)

**Project Overview**
This project analyzes pharmaceutical sales and prescription data using Microsoft Excel to understand product performance, regional trends, and doctor-wise prescribing behavior. The goal is to convert raw data into clear, actionable insights through structured reporting.

**Problem Statement**
Pharma sales data is often large and unstructured, making it difficult to track performance and trends. This project focuses on organizing and analyzing such data to support monitoring and decision-making.

**Objectives**
- Analyze pharmaceutical sales and prescription trends
- Track performance by product, city, and doctor specialty
- Create MIS-style reports for clear business reporting

**Tools & Skills Used**
- Microsoft Excel  
- Pivot Tables  
- Lookup Functions (VLOOKUP / XLOOKUP)  
- Charts and Slicers  
- Data Cleaning and Validation  

**Dataset**
The dataset used in this project is a cleaned, sample pharmaceutical sales and prescription dataset created for learning and portfolio purposes.

**Key Insights**
- A limited number of products contributed to a major share of total sales  
- Certain cities consistently showed higher prescription volumes  
- Doctor specialty had a noticeable impact on prescription patterns  

**Files Included**
- Excel project file (.xlsx)   

## Disclaimer
This project is created for educational and portfolio purposes only. No real or confidential company data has been used.
